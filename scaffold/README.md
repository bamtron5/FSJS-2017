# FSJS-2017 Scaffold

## Includes
 - Setup
  - Package.json Scripts
    - `npm build` with babel from `es2015` to `es5`
    - `npm start`
    - `npm sass` & `npm sass:w` (requires ruby)
 - Client
  - Components
    - MAIN Component
    - Smart Components
    - Presentation Components
  - Directives
    - Socket.io
  - Filters
    - toDate
 - Server
  - Helmet
  - express-session
  - passport local
